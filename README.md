# prefix_tool_web

一个轻量谱面工具，可以进行简单的替换编辑

在线使用：[bugcode.cc/adofai/prefixtool](http://bugcode.cc/adofai/prefixtool/)

工具集主站：[bugcode.cc/adofai](http://bugcode.cc/adofai)

# 布局

采用左中右结构：

| 信息展示 | 谱面文件预览          | 工具集                 |
| ------- | -------------------- | ---------------------- |
| 谱面标签 | 谱面json预览/编辑器   | 事件和对象前缀替换       |
| 事件统计 | 显示冰与火轨道(计划中) | 图片名前缀替换          |
|         |                      | 运动曲线替换Out Elastic |
